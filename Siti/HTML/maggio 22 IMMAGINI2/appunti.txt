CDN: è una rete di server interconnessi che velocizza il caricamento di pagine web ad altro contenuto. Lo scopo principale 
    è ridurre la latenza, sparpagliando le risorse tra i vari server per poi recuperarle quando va in cerca di altre 
    iformazioni richieste dall' utente.
    Alcuni server lo mettomno a diposizione gratis altri posso essere intergrati pagando una somma.

Molto imporate capire ciò che vuole il cliente, facendoil sito sui suoi gusti più che sui nostri.
